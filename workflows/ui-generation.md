# WORKFLOW GIAO DIỆN (UI MODES)

Hệ thống có thanh Tab Navigation để chuyển đổi 3 chế độ [18]:

1. **BREAKDOWN MODE (StudioBinder style) [19]:**
   - Layout: Split Panel (Trái: PDF Text, Phải: Elements Manager) [19].
   - Tương tác: Bôi đen text hiển thị Context Menu chọn màu Cast, Props... theo `style-guide.json` [17].

2. **LINE SCRIPT MODE (Scriptation style) [20]:**
   - Lớp PDF hiển thị ở Center. Bên trái là thanh Lining Tool (Draw Mode, Split Mode, Note, Highlighter) [8, 14, 20, 21].
   - Box Label (Slate Header) nằm đè trên móc vuông, hiển thị Shot# và Shot Size [22].

3. **SHOTLIST MODE (Script Evolution style) [18]:**
   - Bảng Data Table dạng Google Sheets. 4 hàng đầu Frozen, Cột đầu tiên Frozen [23].
   - Cột DESCRIPTION và DIALOGUE phải rộng và cao (~115px) [23].
